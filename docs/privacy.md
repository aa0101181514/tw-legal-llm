# 隱私說明

「法律偵探」(dr-lawbot.com)提供的 TLR 判決檢索服務,為免費公開服務,
不需註冊、不需 API key。本說明描述我們記錄什麼、為什麼記錄、以及如何使用。

## 我們記錄什麼

當你透過 AI 工具使用 TLR 判決檢索時,後端會記錄:

- 你的**查詢內容**(用於改善檢索品質,見下方「用途」)。
- 呼叫的功能(判決搜尋 / 判決內文讀取)。
- 全文讀取請求的判決識別碼(`doc_id`)。
- 回應狀態、延遲與結果數量。
- 時間戳記與粗略來源資訊(用於濫用偵測與服務保護)。

我們**不會**蒐集或要求可識別你個人身分的資訊(姓名、email、帳號等)。
查詢內容以「不綁定個人身分」的方式留存。

## 我們看不到什麼

- 你 AI 工具裡的 prompt、完整對話脈絡。
- 你的 AI(Claude / ChatGPT 等)生成給你的答案。

最終答案只存在你和你的 AI 服務之間,我們完全看不到。

## 查詢內容的用途

我們留存查詢內容,目的是**持續改善判決檢索品質**,具體包括:

- 找出搜尋不到結果或結果不佳的查詢,據以補充資料、調整檢索。
- 統計使用者關心的法律議題分布,規劃資料與內容方向。
- 製作法律主題的文章、FAQ 與判決導讀(SEO / GEO)。

**我們不會把查詢內容用於訓練生成式 AI 模型。** 若未來要將去識別化後的查詢
用於建立檢索評估或標註資料集,也會經過人工審核,並僅限改善本服務的檢索與
資料品質。

## 引用連結

`citation_url` 指向 `dr-lawbot.com/fullview/<doc_id>`,該頁公開渲染判決
原文,也是我們對外發布(Google / ChatGPT / Perplexity 收錄)的同一個
公開 URL。

請求日誌保留供營運與檢索改善使用。我們**不販售、不分享、不對外提供**這些
日誌給第三方。
